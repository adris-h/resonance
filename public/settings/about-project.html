About This Project

Hello!! Before anything, thank you for visiting my app!

So if you’re still confused what this app is about - it’s a music app duh

My main idea was to create sort of a DJ Controller with all the cool features and such, well safe to say I failed miserably but hey, at least I tried. And I'm actually proud of what I managed to do in about 2 months.

This project was created using vanilla JS, Firebase + some other libraries (BPM Detect, ColorPicker), HTML and CSS. The other external resource is the font from Adobe Fonts. Thank you, Adobe for making such a great font available!!!
